# Ingestor Jupyter Notebooks

Interactive tutorials for the ingestor library.

## Notebooks

- **01_quickstart.ipynb**: Get started in 5 minutes
- **02_configuration.ipynb**: Complete configuration guide  
- **03_advanced_features.ipynb**: Advanced techniques
- **04_real_world_legal.ipynb**: Legal document processing
- **05_real_world_medical.ipynb**: Medical device manuals
- **06_troubleshooting.ipynb**: Debug common issues
- **07_performance_tuning.ipynb**: Large-scale optimization

## Setup

```bash
pip install -e ../..
pip install jupyter notebook matplotlib pandas python-dotenv
```

Create `.env` with Azure credentials, then run:

```bash
jupyter notebook
```
