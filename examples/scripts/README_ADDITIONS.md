
### 05_azure_blob.py
**Azure Blob Storage integration**

Process documents stored in Azure Blob Storage and save artifacts to blob storage.

```bash
python examples/scripts/05_azure_blob.py
```

**What it demonstrates:**
- Reading documents from Azure Blob Storage
- Processing files in blob containers
- Glob pattern matching in blob storage
- Saving artifacts to blob storage

### 06_index_management.py
**Index management operations**

Interactive script for managing Azure AI Search indexes.

```bash
python examples/scripts/06_index_management.py
```

**What it demonstrates:**
- Checking index existence
- Validating index configuration
- Creating index backups
- Deploying and updating indexes
- Deleting indexes
- Force recreation with data loss warning
